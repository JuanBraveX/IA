Instrucciones
Para que funcione es necesario instalar:
	*Python:
https://www.python.org/downloads/
	*OpenCV2:
Si ya se tiene Python (cmd):
  	pip install opencv-python
	*Tesseract:
Si ya se tiene Python (cmd):
  	pip install pytesseract

  	*flask
Si ya se tiene Python (cmd):
  	pip install Flask

	*Werkzeug
Si ya se tiene Python (cmd):
	pip install Werkzeug

->32 bits
https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w32-setup-v5.2.0.20220712.exe
->64 bits
https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w64-setup-v5.2.0.20220712.exe
	*Pillow:
Si ya se tiene Python (cmd):
  pip install pillow
	
	*Numpy:
Si ya se tiene Python (cmd):
  pip install numpy

  Para correr se necesita estar en la ruta raiz del index.py y en cmd escribir el 
  comando "python index.py"
  Las rutas pueden variar en donde ejecunten el archivo.
